Instructions for Installing/Starting the Cloud Incubation UI

  1. Download and install node.js --> https://nodejs.org/en/
  2. Download and install GitHub --> https://desktop.github.com/
  3. Open a Gitshell command prompt and browse/create a new folder in which to copy the Cloud UI project
  4. Within Gitshell, create a new root directory in which to store a new project.
  5. Change directory to the new root folder and issue the following command
    • clone ssh://git@gitscm.cisco.com/ccdev/incubation-ui.git
  6. From within the root directory, issue the following commands:
    • npm install
    • node app.js
  7. Browse to the following link: http://localhost:3000/
